목적

크롬 브라우저의 시작 페이지로 활용할 수 있는 개인화된 대시보드를 제작하여, 필요한 정보를 한눈에 볼 수 있도록 구성했습니다.

주요 기능

세계 시계

  • 다중 시간대 동시 표시 (기본: 뉴욕, 런던, 서울)
  • 24시간 형식 지원
  • Local Storage 기반 사용자 설정 저장

Google Trends

  • 실시간/일간 트렌드 토글 기능
  • 트렌드별 상세 뉴스 확장/축소
  • 5분 주기 자동 업데이트
  • 전체 뉴스 동시 확장/축소

시장 지표

  • 주요 지표 실시간 모니터링 (나스닥, S&P 500, 다우존스, VIX)
  • 미국 국채 수익률 추적 (5년, 10년, 30년)
  • 차트 시각화 및 등락률 색상 구분
  • 5분 주기 자동 업데이트

기술 스택

백엔드

  • FastAPI - 현대적인 Python 웹 프레임워크
  • Uvicorn - ASGI 서버
  • yfinance - 주식 데이터 API
  • httpx - 비동기 HTTP 클라이언트

프론트엔드

  • TailwindCSS - 유틸리티 우선 CSS 프레임워크
  • Chart.js - 데이터 시각화
  • Local Storage - 클라이언트 측 데이터 저장

보안 설계

CORS 정책

  • 화이트리스트 기반 출처 제한
  • HTTP 메서드 제한 (GET only)
  • 정적 파일 접근 제한
